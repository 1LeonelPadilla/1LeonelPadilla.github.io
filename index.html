1Html

HTML (Lenguaje de Marcado de Hipertexto) es un lenguaje de programación que se utiliza para crear y estructurar páginas web. Al escribir en HTML, se pueden agregar elementos como encabezados, párrafos, imágenes, enlaces y listas a una página web. HTML también da significado y estructura a los contenidos de la página para que los navegadores y otros dispositivos puedan comprender y presentar la información de manera adecuada. Conocer HTML es fundamental para cualquier persona que desee desarrollar sitios web.

2CSS

CSS (Hojas de Estilo en Cascada) es un lenguaje de estilo que se utiliza para dar estética y formato a las páginas web creadas en HTML. Con CSS, se pueden cambiar cosas como el color de fondo, la fuente, el tamaño y la disposición de los elementos en una página web. CSS permite separar la presentación de la estructura de una página web, lo que facilita la modificación y mantenimiento de la apariencia de un sitio. Además, CSS permite la creación de estilos reutilizables para aplicar a múltiples páginas en un sitio web, lo que ahorra tiempo y esfuerzo.

3Javascript

JavaScript es un lenguaje de programación de alto nivel que se utiliza en el desarrollo de aplicaciones y sitios web interactivos. Con JavaScript, se pueden agregar características dinámicas y efectos visuales a una página web, tales como animaciones, formularios interactivos, juegos y mucho más. JavaScript también permite a los desarrolladores validar formularios en el lado del cliente y realizar operaciones y cálculos en tiempo real. Además, JavaScript es un lenguaje de programación versátil que se ejecuta en una amplia variedad de plataformas, incluyendo navegadores web, servidores y dispositivos móviles.
